# Nili Baby - עיצוב ממשק משתמש

## סקירה כללית

האפליקציה עוצבה בסגנון מודרני וורוד קלאסי, עם דגש על חוויית משתמש נעימה להורים חדשים.

## מסכים

### 1. מסך פתיחה (Splash Screen)
- **קובץ:** `src/screens/SplashScreen.tsx`
- **תיאור:** מסך פתיחה אנימטיבי עם לוגו האפליקציה
- **אנימציות:**
  - Fade in של הלוגו
  - Scale animation
  - סיבוב של אייקון הבקבוק
  - הופעה הדרגתית של פיצ'רים
- **משך:** 3 שניות

### 2. מסך התחברות/הרשמה (Login Screen)
- **קובץ:** `src/screens/LoginScreen.tsx`
- **תיאור:** מסך התחברות עם טאבים להחלפה בין התחברות והרשמה
- **עיצוב:**
  - רקע ורוד עם עיגולים דקורטיביים
  - טופס בכרטיס לבן מעוגל
  - אייקונים בשדות הקלט
  - אנימציות מעבר

### 3. מסך הבית (Home Screen)
- **קובץ:** `src/screens/HomeScreen.tsx`
- **תיאור:** מסך ראשי עם סיכום מצב התינוק
- **רכיבים:**
  - כותרת עם שם התינוק וגיל
  - טיימר להאכלה הבאה
  - סטטיסטיקות יומיות
  - כרטיס רישום האכלה
  - כפתורי פעולה מהירה

### 4. מסך היסטוריית האכלות (Feeding History Screen)
- **קובץ:** `src/screens/FeedingHistoryScreen.tsx`
- **תיאור:** רשימת האכלות עם סטטיסטיקות
- **רכיבים:**
  - כרטיס סטטיסטיקות (7 ימים)
  - רשימת האכלות עם אייקונים צבעוניים
  - אפשרות מחיקה

### 5. מסך תזכורות (Reminders Screen)
- **קובץ:** `src/screens/RemindersScreen.tsx`
- **תיאור:** ניהול תזכורות יומיות
- **רכיבים:**
  - כותרת עם מספר תזכורות פעילות
  - רשימת תזכורות עם מתג הפעלה
  - תזכורות מהירות מוגדרות מראש
  - מודל להוספת תזכורת

### 6. מסך תורים (Appointments Screen)
- **קובץ:** `src/screens/AppointmentsScreen.tsx`
- **תיאור:** ניהול תורים לרופאים וטיפת חלב
- **רכיבים:**
  - כותרת עם מספר תורים קרובים
  - רשימת תורים עם אייקונים
  - כפתורי הוספה מהירה
  - מודל להוספת תור

### 7. מסך הגדרות (Settings Screen)
- **קובץ:** `src/screens/SettingsScreen.tsx`
- **תיאור:** הגדרות האפליקציה והתינוק
- **סקציות:**
  - הגדרות האכלה (מרווח וכמות יעד)
  - שיתוף עם בן/בת זוג
  - ניהול תינוקות
  - התנתקות

### 8. מסך הוספת תינוק (Add Baby Screen)
- **קובץ:** `src/screens/AddBabyScreen.tsx`
- **תיאור:** טופס להוספת תינוק חדש
- **רכיבים:**
  - אנימציה של אייקון תינוק
  - טופס עם שדות שם ותאריך לידה
  - רשימת פיצ'רים

## קומפוננטים

### NextFeedingTimer
- טיימר אנימטיבי להאכלה הבאה
- אפקט פעימה כשמתקרב הזמן
- התראה קולית 10 דקות לפני

### FeedingHistoryItem
- כרטיס האכלה עם אייקון צבעוני
- הבחנה בין הנקה לתמ"ל
- אנימציית לחיצה

### ReminderItem
- כרטיס תזכורת עם מתג
- אייקון אוטומטי לפי סוג התזכורת
- אנימציית לחיצה

### AppointmentItem
- כרטיס תור עם פס צבעוני
- אייקון אוטומטי לפי סוג התור
- תג "קרוב" לתורים קרובים

## אנימציות

האפליקציה משתמשת ב-React Native Animated API עבור:
- Fade in/out
- Scale animations
- Slide animations
- Pulse effects
- Rotation
